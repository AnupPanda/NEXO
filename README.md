# NEXO

NEXO is a free universal file converter web app built with FastAPI, HTML, CSS, and JavaScript.

## Features
- DOCX to PDF
- PDF to DOCX
- JPG to PDF
- PNG to PDF
- PDF to JPG
- PDF to PNG
- JPG to PNG
- PNG to JPG
- Merge multiple images into one PDF

## Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI
- Libraries: Pillow, PyMuPDF, pdf2docx, python-multipart
- DOCX to PDF uses LibreOffice headless
